Using ``text_indentation``
-------------------

Import:

    >>> text_indentation = __import__('5-text_indentation').text_indentation

Now use it:


    >>> text_indentation('hello. world how are you')
    hello.
    <BLANKLINE>
    world how are you

    >>> text_indentation('hello? world?')
    hello?
    <BLANKLINE>
    world?
    <BLANKLINE>

    >>> text_indentation('hello:   world')
    hello:
    <BLANKLINE>
    world
